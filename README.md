# Cosa

Layout tastiera italiano con facile accesso alla schwa (ə) e schwa lunga (ɜ), per Mac.

# Come installare

Scarica lo zip dalle [release](https://github.com/Sbax/layout-italiano-inclusivo-mac-o-sx/releases/latest).\
Sposta il file `Italiano Inclusivo.bundle` nella cartella `~/Library/Keyboard Layouts` e riavvia il Mac.\
Per trovare la cartella rapidamente apri il terminale e digita `open ~/Library/Keyboard\ Layouts`.

Una volta riavviato aggiungi il layout dalle Impostazioni di Sistema:
- Clicca su Apple () > Preferenze di Sistema, poi su Tastiera, clicca Modifica... sulla riga Sorgenti di input.
- Clicca sul bottone "+", seleziona "Italiano" nel menu a sinistra
- Seleziona il layout "Italiano Inclusivo" e clicca su aggiungi
- Una icona con la bandiera italiana apparirà nella barra dei menu, in alto a destra nello schermo. Clicca sulla bandiera per scegliere il nuovo layout

# Come digitare

Per digitare la schwa per il singolare (ǝ) `option (⌥) + a`. \
Per digitare la schwa lunga per il plurale (ɜ) `option (⌥) + i`.

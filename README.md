# Cosa

Layout tastiera italiano con facile accesso alla schwa (ə) e schwa lunga (ɜ), per Mac.

# Come installare

Sposta il file `Italiano Inclusivo.bundle` nella cartella `~/Library/Keyboard Layouts` e riavvia il Mac.\
Per trovare la cartella rapidamente apri il terminale e digita `open ~/Library/Keyboard\ Layouts`.

# Come digitare

Per digitare la schwa per il singolare (ǝ) `option (⌥) + a`. \
Per digitare la schwa lunga per il plurale (ɜ) `option (⌥) + i`.
